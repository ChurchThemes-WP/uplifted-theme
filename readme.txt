** Uplifted Theme **

Uplifted is a WordPress theme for churches that requires the Church Theme Content plugin to function properly:

https://wordpress.org/plugins/church-theme-content/

Documentation for the Uplifted Theme can be found here:

https://churchthemes.net/getting-started-uplifted/

For questions, please contact support@churchthemes.net. Please make sure you have purchased a theme support license to ensure a quick turnaround for your issue. A theme license may be purchased from our website:

https://churchthemes.net/themes/uplifted-theme/
